Simulador de Patrimonio

Hipótesis del modelo

Este simulador utiliza datos basados en el comportamiento histórico de los mercados financieros globales.

  • Rentabilidad media histórica de acciones globales: ~7-10 % anual
  • Rentabilidad media cartera diversificada: ~6-8 % anual
  • Inflación media histórica economías desarrolladas: ~2-3 %

Para mantener un modelo conservador se utiliza una rentabilidad media del 7 % y una inflación media del 2.5 %.

Objetivo del Fondo de Patrimonio Familiar

El objetivo del fondo es preservar y hacer crecer el patrimonio familiar a largo plazo mediante inversión disciplinada y diversificada.

No busca especulación ni beneficios rápidos, sino estabilidad y crecimiento sostenido durante décadas.

  • Protección frente a la inflación
  • Crecimiento del capital familiar
  • Horizonte multigeneracional
  • Disciplina financiera

Filosofía de inversión

  • Diversificación global
  • Costes bajos
  • Horizonte de inversión largo
  • Rebalanceo periódico
  • Disciplina frente a ciclos económicos

Este sistema está diseñado para funcionar durante más de 100 años, permitiendo que el patrimonio se beneficie del poder del interés compuesto.

Interpretación de los parámetros

Capital inicial Cantidad de dinero con la que empieza el fondo.

Aportación anual Cantidad añadida al fondo cada año.

Rentabilidad esperada Media histórica aproximada de una cartera diversificada global.

Inflación Pérdida media anual del poder adquisitivo del dinero.

Años Horizonte temporal de la inversión.
